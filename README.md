## Local Installation and Setup Instructions
Clone down this repository. You will need `node` and `npm` installed globally on your machine.

1) Installation:
```npm install```

2) To Run Test Suite:
```npm test```

3) To Start Server:
```npm start```

4) To Visit App in browser:
http://localhost:8000/home
